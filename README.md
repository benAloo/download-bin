# Downloading bin items like winutils.exe and hadoop.dll etc

### Clone the repo

```bash
git clone https://github.com/benAloo/download-bin.git
```

### Create a virtual environment

```bash
python -m venv .venv
```

### Activate virtual environment (Windows)

```bash
.venv\Scripts\Activate
```

### Install dependencies

```bash
pip install requests
```
